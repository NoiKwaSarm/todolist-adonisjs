## Project Setup

```sh
npm install
```

### For Development

```sh
npm run dev
```

### For Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

### Start Docker Compose - Postgresql

```sh
npm run start:pg
```

### Stop Docker Compose - Postgresql

```sh
npm run stop:pg
```

### Run Migration Files

```sh
npm run migrate:run
```

### Postman Testing API
Import file: todolist.postman_collection.json

Set environment with url

http://localhost:3333
