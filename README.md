## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

### Start Docker Compose - Postgresql

```sh
npm run start:pg
```

### Stop Docker Compose - Postgresql

```sh
npm run stop:pg
```

### Run Migration Files

```sh
npm run migrate:run
```
